## Appium Setup
In order to run camelQA scripts, you'll need to setup Appium on your machine. If you're new to Appium, check out these guides on how to get setup.

<AccordionGroup>
  <Accordion icon="rectangle-terminal" title="Physical iOS Devices">
    If you'd like to get started using the iOS simulator instead, you do not need to configure Web Driver Agent.
    Read this post on [setting up appium for iOS devices](https://medium.com/tauk-blog/setup-guide-for-appium-real-ios-device-e35eabbf630e) on Medium.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Physical Andriod Devices">
    Read this post [setting up appium for Andriod devices](https://www.browserstack.com/guide/run-appium-android).
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Andriod Studio">
    Read this post [setting up appium for Andriod simulators](https://anivaz.medium.com/how-to-set-up-appium-with-android-studio-for-mobile-app-testing-a300b7910364).
  </Accordion>
  
</AccordionGroup>
